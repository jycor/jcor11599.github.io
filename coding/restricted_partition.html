Restricted Partition

How many different ways to divide N people into K groups, where the size(group[i]) >= size(group[i-1])?

p(n,k) = number of ways to partition n into k parts.

First, put 1 person in each group; this leaves us with n - k people left.

p(n,k) = p(n-k,1) + p(n-k,2) + ... + p(n-k,k-1) + p(n-k,k)

Note that,

p(n-1,k-1) = p(n-k,1) + p(n-k,2) + ... + p(n-k,k-1) + p(n-k,k-1)

As a result, we get the recurrence

p(n,k) = p(n-1,k-1) + p(n-k,k)

Python3 code:

    def algorithm(n, k):
        if n < k:
            return 0
        dp = [[-1 for i in range(200)] for j in range(200)] # 200x200 matrix
        for i in range(1, k+1):
            for j in range(1, n+1):
                if i == j:
                    dp[i][j] = 1;
                else:
                    dp[i][j] = dp[i-1][j-1] + dp[i][j-k]
        return dp[k][n]
